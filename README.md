# Mavi Palestine Database Management System

A full-featured database management system developed for the local **Mavi Jeans branch in Palestine**, designed to streamline retail operations and improve efficiency across sales, purchasing, payments, and inventory management.

---

## 📌 Overview

This system provides a centralized platform for managing day-to-day retail activities. It enables accurate tracking of products, financial transactions, and stock levels while supporting data-driven decision-making.

The project reflects practical implementation of database design concepts such as normalization, relational modeling, and optimized querying.

---

## 🚀 Features

- 🛒 **Sales Management**
  - Record and track customer purchases
  - Generate sales history and reports

- 📦 **Inventory Management**
  - Real-time stock tracking
  - Monitor product quantities by size, color, and location

- 🧾 **Purchase Management**
  - Manage supplier orders
  - Track incoming inventory

- 💳 **Payment Tracking**
  - Record and manage financial transactions
  - Monitor cash flow and payment status

- 🔍 **Data Insights**
  - Generate reports for better decision-making
  - Analyze sales trends and stock movement

---

## 🏗️ System Design

The database is designed using relational models and follows normalization principles to ensure:
- Data consistency
- Reduced redundancy
- Efficient querying

### Main Entities:
- Store
- Employee
- Supplier
- Product (with size, color, cost)
- Inventory
- Sales & Sales Details
- Purchases & Purchase Details
- Payments
- Stock Transfers
